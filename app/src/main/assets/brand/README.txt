Middin wordmark (in-app + launcher)
===================================

The app uses a single raster wordmark with **no Compose tinting** or color filters:

  app/src/main/res/drawable-nodpi/middin_wordmark.png

To update the logo:

1. Export your official wordmark as PNG (preserve transparency if applicable).
2. Replace the file above **without renaming** (`middin_wordmark.png`).
3. Rebuild the app.

The launcher adaptive icon uses the same PNG (centered on a white background) so the mark matches everywhere.
