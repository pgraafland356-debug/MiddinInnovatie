Middin Innovatie — offline gebruikers (Android + Windows)
=======================================================

Bewerk dev-accounts.json en voeg een regel toe aan "accounts":

  { "username": "jan", "password": "jouw-wachtwoord" }

Daarna synchroniseren:

  .\scripts\sync-dev-accounts.ps1

Admin (instellingen / feed / Gemini): alleen pieter-bas (zie ENDPOINT_SETTINGS_USERNAME in code).
